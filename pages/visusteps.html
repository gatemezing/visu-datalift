Wizard de visualisation Datalift

Ce wizard vous aidera à créer des visualisations appropriées à votre jeu de données, en utilisant la plate-forme Datalift

  • Dataset
  • Choix du widget de visualisation
  • Exemple

1. Dataset (Jeu de données)

Avant de commencer à utiliser le wizard, il est important d'indiquer le jeu de données qui servira à créer une ou des applications associées, de préférence des données en RDF enrichies d'autres données. Un exemple de jeux de données que nous pensons d'un certain intérêt est disponible à cette adresse

2. Choix du widget

Nous analysons votre jeu de données pour déterminer s'il contient de la géolocalisation, des personnes, des adresses, des séries temporelles, des prix, etc. En fonction de cette analyse, nous vous préconisons le type de visualisation adapté à votre jeu de donnés.

3. Exemples de visualisation

Nous présentons ici des exemples de visualisation utilisant des données liées du web de données faites par les partenaires de datalift.

  • Event Media

    Une application aggrégeant les médias et les évènements en ligne, réalisée par Eurécom.

  • LOV Dataset

    Une application du catalogue de données développée par Mondeca

  • Election 1er tour en France

    Une application issue des données "liftées" par la plate-forme Datalift permettant de voir non seulement les scores des candidats, mais aussi le taux de chômage dans les départements, la description des candidats, etc. lors du premier tour des élections présidentielles en France en 2012.

4. Template HTML de base

En général, les applications de visualisation utilisent un fichier html type. Dans ce fichier, il peut faire appel à des scripts ou des fichiers de style ou CSS.

Voici un exemple de fichier type pour une application :

              <!DOCTYPE html>
              <html>
                <head>
                  <title>Titre test template HTML</title>
                </head>
                <body>
                  <h1>Hello, Datalift!</h1>
                  <script src="http://code.jquery.com/jquery-latest.js"></script>
                </body>
              </html>

Pour faire de ce fichier un template Exhibit, il faut indiquer les fichiers CSS et JS.:

              <!DOCTYPE html>
              <html>
                <head>
                  <title>Titre test Exhibit Template</title>
                  <link href="ly-data.js" type="application/json" rel="exhibit/data" />
                  <script scr="http://static.simile.mit.edu/exhibit/api-2.0/exhibit-api.js" 
                  type="application/json"></script>
                  <!-- Exhibit template -->
                  <style> /* ajouter le style ici */ <style>
                </head>
                <body>
                  <h1>Hello, Datalift!</h1>
                  <script src="http://code.jquery.com/jquery-latest.js"></script>
                  
                </body>
              </html>

And you're set! With those two files added, you can begin to develop any site or application with Bootstrap.
